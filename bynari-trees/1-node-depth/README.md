## Question
Writes a function to sum the node depth of a binary tree
for e.g:
input: 1
      / \
     2   3
    / \ / \
   4  5 6  7
  / \
 8   9

output:
16
// depth of the value 2 is 1.
// depth of the value 3 is 1.
// depth of the value 4 is 2.
// Summing up all the depths end up with the value 16.
